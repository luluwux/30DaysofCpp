# Simple C++ To-Do List

This is a simple to-do list application that runs in your terminal. It was built using C++.

## Features

  * **Add Tasks:** Add new tasks to your list.
  * **Remove Tasks:** Delete tasks you no longer need.
  * **Complete Tasks:** Mark tasks as done.
  * **View Tasks:** See all your current tasks and their status (✅ or ❎).

## How to Run

1.  **Get the Code:**
    Download or clone the project files.

2.  **Compile:**
    You need a C++ compiler (like `g++`). Open your terminal in the project folder and run:

    ```bash
    g++ main.cpp -o todo
    ```

    *(Note: This assumes your main file is named `main.cpp` and your helper file `Helper.h` is correctly included.)*

3.  **Run:**
    After compiling, run the program:

    ```bash
    ./todo
    ```

    *(On Windows, you might type `todo.exe`)*

## Tech Used

  * **C++**
  * **Standard Library:** `<iostream>`, `<vector>`, `<string>`
